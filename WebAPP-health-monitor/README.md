# WebAPP 健康巡检与自愈系统

一个基于 Python 的自动化运维监控工具，用于监控 Web 应用及基础设施的健康状态，并在发现故障时自动尝试修复，保障服务高可用。

## ✨ 功能特性

- ✅ **Web 服务健康检查**：HTTP 状态码检测、响应时间统计
- ✅ **端口存活检测**：TCP 端口连通性检查
- ✅ **进程状态监控**：进程存在性检查
- ✅ **系统资源监控**：CPU、内存、磁盘使用率监控
- ✅ **故障自动修复**：服务异常时自动重启，实现自愈
- ✅ **防雪崩设计**：重试次数限制，避免无限重启
- ✅ **结构化日志**：按天切割日志，便于排查问题
- ✅ **健康报告生成**：支持 Markdown/TXT 格式报告
- ✅ **配置与代码分离**：所有可配置项独立管理

## 🛠️ 技术栈

- **语言**：Python 3
- **核心库**：
  - `psutil` - 系统资源监控
  - `requests` - HTTP 请求
  - `PyYAML` - 配置文件解析
  - `logging` - 日志管理（标准库）
  - `socket` - 网络检查（标准库）

## 📁 项目结构
- logger.py：日志模块（控制台+文件+轮转）
- health_checker.py：健康检查（Web/端口/进程/系统资源）
- auto_repair.py：自动修复（重试+防雪崩）
- config.yaml：配置中心
- monitor.py：主调度入口
- logs/：日志目录
- reports/：报告输出
