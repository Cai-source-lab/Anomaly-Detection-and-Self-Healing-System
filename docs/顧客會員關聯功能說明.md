# 顧客與會員關聯功能說明

## 概述

已成功實施方案A：在顧客表中添加會員ID外鍵，實現顧客與會員的關聯功能。

## 數據庫變更

### 1. 執行遷移腳本

請先執行數據庫遷移腳本：
```sql
-- 文件位置：docs/sql/add_customer_vip_relation.sql
```

該腳本會：
- 在 `cy_customer` 表中添加 `vip_id` 字段
- 創建索引以提升查詢效率
- 添加外鍵約束，確保數據完整性

### 2. 表結構變更

`cy_customer` 表新增字段：
- `vip_id` INT UNSIGNED NULL - 關聯會員ID（可為空）

## 代碼變更

### 1. 實體類變更

`cy_customer.java` 新增：
- `vipId` 字段（Integer類型，可為空）
- `getVipId()` 和 `setVipId()` 方法

### 2. DAO層新增方法

`cy_customerDao.java` 新增以下方法：

#### 2.1 根據會員ID查詢顧客列表
```java
List<cy_customer> getCustomersByVipId(int vipId)
```
用途：查詢某個會員關聯的所有顧客

#### 2.2 關聯查詢顧客及會員信息
```java
cy_customer getCustomerWithVipInfo(int customerId)
```
用途：查詢顧客時同時獲取關聯的會員信息

#### 2.3 更新顧客的會員關聯
```java
boolean updateCustomerVipRelation(int customerId, Integer vipId)
```
用途：為顧客設置或更新會員關聯
- `vipId` 為 null 時表示解除關聯

#### 2.4 解除顧客的會員關聯
```java
boolean removeCustomerVipRelation(int customerId)
```
用途：移除顧客的會員關聯

#### 2.5 根據電話自動匹配會員
```java
boolean autoLinkCustomerToVipByPhone(int customerId)
```
用途：根據顧客電話號碼自動匹配並關聯會員（如果電話相同）

#### 2.6 獲取顧客關聯的會員信息
```java
cy_Hy getCustomerVipInfo(int customerId)
```
用途：獲取顧客關聯的完整會員信息（包括餘額、積分等）

### 3. 現有方法更新

以下方法已更新以支持 `vip_id` 字段：
- `getCustomerByCode(String customerCode)`
- `getCustomerById(int customerId)`

## 使用示例

### 示例1：為顧客關聯會員
```java
cy_customerDao customerDao = new cy_customerDao();

// 將顧客ID=1關聯到會員ID=5
boolean success = customerDao.updateCustomerVipRelation(1, 5);
if (success) {
    System.out.println("關聯成功");
}
```

### 示例2：根據電話自動匹配會員
```java
cy_customerDao customerDao = new cy_customerDao();

// 自動匹配顧客ID=1的會員（根據電話號碼）
boolean success = customerDao.autoLinkCustomerToVipByPhone(1);
```

### 示例3：查詢顧客的會員信息
```java
cy_customerDao customerDao = new cy_customerDao();

// 獲取顧客ID=1關聯的會員信息
cy_Hy vip = customerDao.getCustomerVipInfo(1);
if (vip != null) {
    System.out.println("會員名稱：" + vip.getVname());
    System.out.println("帳戶餘額：" + vip.getQb());
    System.out.println("積分：" + vip.getJf());
}
```

### 示例4：查詢某會員的所有顧客
```java
cy_customerDao customerDao = new cy_customerDao();

// 查詢會員ID=5關聯的所有顧客
List<cy_customer> customers = customerDao.getCustomersByVipId(5);
for (cy_customer customer : customers) {
    System.out.println("顧客名稱：" + customer.getCustomerName());
}
```

### 示例5：解除關聯
```java
cy_customerDao customerDao = new cy_customerDao();

// 解除顧客ID=1的會員關聯
boolean success = customerDao.removeCustomerVipRelation(1);
```

## 業務場景應用

### 場景1：結算時使用會員餘額
```java
// 1. 獲取顧客的會員信息
cy_Hy vip = customerDao.getCustomerVipInfo(customerId);

// 2. 檢查會員餘額
if (vip != null) {
    double balance = Double.parseDouble(vip.getQb());
    if (balance >= totalAmount) {
        // 使用會員餘額支付
        // ... 扣款邏輯
    }
}
```

### 場景2：結算時累積會員積分
```java
// 1. 獲取顧客的會員信息
cy_Hy vip = customerDao.getCustomerVipInfo(customerId);

// 2. 累積積分
if (vip != null) {
    int currentPoints = Integer.parseInt(vip.getJf());
    int newPoints = currentPoints + (int)(totalAmount / 10); // 每10元1積分
    // ... 更新積分邏輯
}
```

### 場景3：新顧客註冊時自動關聯
```java
// 創建新顧客後，根據電話自動匹配會員
int newCustomerId = ...; // 新創建的顧客ID
customerDao.autoLinkCustomerToVipByPhone(newCustomerId);
```

## 注意事項

1. **外鍵約束**：刪除會員時，相關顧客的 `vip_id` 會自動設置為 NULL（ON DELETE SET NULL）
2. **空值處理**：`vip_id` 可以為 NULL，表示顧客未關聯會員
3. **數據一致性**：外鍵約束確保 `vip_id` 只能引用存在的會員ID
4. **電話匹配**：自動匹配功能基於電話號碼，確保電話號碼格式一致
